Ribbon Menu Deep Dive

The Ribbon is your command center. Click on the tabs below to explore the tools and functions available in each category.

Project Management

Manages project-level settings. Here you can create new projects, open existing files, and manage global project parameters.

Core Modeling Tools

This is where connections are built. Key tools include adding structural Members and Plates, applying manufacturing Operations like welds and bolts, and accessing the Connection Library for reusable templates.

Analysis & Verification

Contains tools to run various analyses on your model. You can perform Stress/Strain checks, Stiffness analysis, and Buckling analysis. The main 'Calculate' button initiates the finite element analysis.

Documentation & Output

Generates detailed calculation reports, bills of material, and drawings. Reports are highly customizable and can be exported to standard formats like PDF and DOC for submittals.

Material Definitions

Defines and manages the material grades for all components in the project, including steel plates, bolts, and concrete blocks, ensuring calculations are based on the correct material properties.

Advanced Customization

Provides access to advanced features such as creating parametric designs, interacting with the software's API for custom scripts, and other automation tools for power users.

Analysis & Reporting

Get clear, code-based pass/fail results in minutes and generate comprehensive reports to document your work.

Instant Results

The software provides a clear summary of all code checks. In the example chart, you can see a typical distribution of analysis results for plates, bolts, and welds, making it easy to spot which components have passed, are close to their limit, or have failed.

Efficient Modeling Features

Speed up your design process with powerful templates and a collaborative connection library.

Parametric Templates

Start with over 250 pre-built, editable templates for common connections, saving significant time on initial setup.

Publish to Library
