What You'll Learn

Bolted & Welded Connections

Master the techniques for modeling precise and efficient bolted and welded steel connections.

AISC Code Compliance

Ensure your designs meet the rigorous standards of the American Institute of Steel Construction.

Moment & Pinned Connections

Deep dive into the design and analysis of the most common and critical connection types.

Base Plate Design

Learn the fundamentals of designing robust and stable base plates for various column types.

Parametric Modeling

Develop versatile models that can be easily adapted to a wide range of project requirements.

CBFEM-Powered Analysis

Leverage the power of Component-Based Finite Element Method for accurate and fast analysis.

Course Curriculum

Module 1: Introduction

  • Fundamentals of steel connections
  • Types of steel connections
  • Code-based design overview
  • Connection failure modes
  • Introduction to IDEA StatiCa

Module 2: Getting Started

  • User Interface Walkthrough
  • Overview of templates & libraries
  • The workflow: Model > Load > Check > Report

Module 3: Modeling Connections

  • Creating connections with templates
  • Manual modeling of custom joints
  • Setting boundary conditions
  • Assignment 1: Custom modeling

Module 4: BIM Integration

  • Workflow with STAAD.Pro
  • Using the IDEA StatiCa Checkbot

Module 5: Load Application

  • Types of loads and combinations
  • Applying forces and moments
  • Significance of load positioning

Module 6: Code Checking

  • Design checks as per AISC
  • Interpreting results
  • Addressing errors and warnings
  • Buckling analysis interpretation

Module 7: Advanced Features

  • Parametric modeling
  • Weld and bolt detailing
  • Plate buckling significance

Module 8: Real-World Examples

  • Extended moment connections
  • Bracing with gusset plates
  • Base plate design
  • Complex multi-member joints

Module 9: Documentation

  • Generating design reports
  • Customizing report content
  • Exporting drawings & snapshots
